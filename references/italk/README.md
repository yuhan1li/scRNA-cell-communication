# iTALK 分析指南

## 方法简介

iTALK 是一个简单直观的细胞通讯可视化工具，专注于快速展示配体-受体互作网络。适合初步探索或快速生成示意图。

**注意**：iTALK **不提供统计显著性检验**，结果仅供初步参考，不能用于发表。

## 输入文件

| 文件 | 格式 | 说明 |
|------|------|------|
| Seurat 对象 | `.rds` | 包含差异基因分析结果 |

## 分析流程

1. **差异基因分析** → 找各细胞类型的 marker 基因
2. **提取配体/受体** → 从 marker 基因中筛选
3. **构建网络** → 随机采样配体-受体对
4. **可视化** → igraph 网络图

## 输出结果

| 文件 | 内容 |
|------|------|
| 网络图 | 配体-受体互作网络（igraph）|

## 适用场景

- ✅ 快速初步探索，无需复杂设置
- ✅ 生成示意图用于内部讨论
- ✅ 教学演示

## 注意事项

- ❌ **不提供统计检验**，结果不能发表
- ❌ 互作对是随机采样，不代表真实通讯
- ✅ 建议后续用 CellChat 或 Cellphonedb 验证
